Why Use CSS?

CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.

What is CSS?New

  • CSS stands for Cascading Style Sheets 1
  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media 2
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once 3
  • External stylesheets are stored in CSS files 4
